## Python
Python version 3.12

## Handler
Handler for Python is defined as
- Python file name
- . delimiter
- method name


## Code Bundle
A zip file containing all the python file(s).

### Example 
Typical zip structure
```
|-- uplus_function.py
```


## Example
A python function defined in uplus_function.py that accepts 2 parameters
``` python
def compute(rhs: int, lhs: int):
    return {
        ...
    }
```

LaunchPad handler for the function
```
uplus_function.compute
```
